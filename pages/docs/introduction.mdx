## Overview
**Cerberusui** is a [react](https://react.dev/) component-library inspired by the cerberus device from the movie **Olympus Has Fallen** and the interceptor system from the movie **Interceptor**.

![cerberus device](/Cerberus-Olympus-has-fallen.jpg)
Cerberus System from the Movie **Olympus Has Fallen**. Source: [Has Fallen Wiki](https://has-fallen.fandom.com/wiki/Cerberus)

![interceptor system](/Interceptor.PNG)
Interceptor System from the Movie **Interceptor**. Source: **Interceptor** at 1h 23min

## Installation
You need [Node.js](https://nodejs.org/en) installed and a React based project like [create-react-app](https://create-react-app.dev/), [Vite](https://v3.vitejs.dev/) or [Next.js](https://nextjs.org/) setup.
```sh npm2yarn copy
npm i cerberusui
```
